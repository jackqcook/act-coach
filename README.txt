# ACT Tutor Web Application

This is a web application that uses OpenAI models to serve as an ACT tutor. The application is built using TypeScript, React for the frontend, and Node.js for the backend.

## Getting Started

### Prerequisites

- Node.js
- npm
- OpenAI API Key

### Setup

1. Clone the repository.
2. Navigate to the `act-tutor-frontend` directory and run:
   ```
   npm install
   npm start
   ```
3. Navigate to the `act-tutor-backend` directory and run:
   ```
   npm install
   node index.js
   ```

### Usage

- Open your browser and go to `http://localhost:3000`.
- Ask questions related to the ACT, and get responses powered by OpenAI.
